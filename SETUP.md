# FitBook bootstrap (ЛР0–ЛР1)

## 1) Встановити
- Node.js 20+
- pnpm (через corepack)

## 2) Встановити залежності
```bash
pnpm install
```

## 3) Запуск
```bash
pnpm dev
```
- Web: http://localhost:3000
- API: http://localhost:5000/health

## 4) Перевірки
```bash
pnpm format
pnpm lint
pnpm test
pnpm build
```

## 5) Conventional commits
Приклад:
- chore: bootstrap fitbook monorepo
- feat(web): add sessions page
- feat(api): add sessions controller
